## 躓きポイントと即解決策

- **Node.js バージョン不足**  
  `node -v` / `npm -v` を確認。**Node 22+** が必須。
- **依存インストール失敗**  
  企業／学内プロキシ環境では `npm config set proxy` を案内。
- **フォーム送信でページがリロード**  
  `e.preventDefault()` を忘れがち → 実演して理由を説明。
- **イベントが後から追加した要素に届かない**  
  イベント委任の概念を図解でフォロー。
